EXPERIENCE

SOCAL LAB(Current)

I am currently working in a Computer Architecture lab under Professor Wong. In this lab, I am working on a custom cooling system for a Nvidia GPU using both software and hardware related technologies. I will be using the CUDA language to program loads onto the GPU(A Nvidia GTX 950). In addition we will use a FLIR camera to analyze the temperature when the GPU is under load and then use our custom built cooling system to cool down the GPU.
